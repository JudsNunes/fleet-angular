# Plan: Create `.mimocode/mimocode.json` with permission rules

## Goal
Create the MiMoCode project config file with edit permission rules requiring confirmation for specific directories.

## Target file
`.mimocode/mimocode.json`

## Content
```jsonc
{
  "$schema": "https://mimo.xiaomi.com/mimocode/config.json",
  "permission": {
    "edit": {
      "src/**": "ask",
      "openspec/**": "ask",
      ".mimocode/skills/**": "ask"
    }
  }
}
```

## What this does
- Adds `$schema` for editor validation
- Configures `edit` tool to require user confirmation ("ask") when modifying files under:
  - `src/**` — application source code
  - `openspec/**` — spec files
  - `.mimocode/skills/**` — custom skills

## Verification
1. Open `.mimocode/mimocode.json` after creation
2. Confirm valid JSON structure
3. Edit a file under `src/` and verify the TUI prompts for permission
